# 🛒 InsightEngine – E-Commerce Customer Analytics Dashboard

> Code A Nova Internship Project – Data Science

InsightEngine is an end-to-end data science project focused on analyzing e-commerce customer behavior. The project aims to transform raw transactional data into meaningful business insights through data preprocessing, exploratory data analysis, customer segmentation, churn prediction, and a recommendation system.

---

# 📌 Project Objective

The objective of this project is to perform the complete data science lifecycle on an e-commerce dataset by:

- Cleaning and preprocessing customer purchase data
- Performing Exploratory Data Analysis (EDA)
- Engineering meaningful customer features
- Segmenting customers using RFM Analysis & K-Means Clustering
- Building a customer churn prediction model
- Creating a product recommendation engine
- Presenting all insights using an interactive Streamlit dashboard
